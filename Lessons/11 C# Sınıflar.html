Sınıflar

Sınıflar Nedir?

Sınıflar, fonksiyonları veya değişkenleri gruplandırabilceğimiz ve toplu bir şekilde değerlendirebilceğimiz yapılarıdır.
İleride görüceğimiz OOP yani obje yönelimli programlama da sınıflar ana odaktır.

Nesne Oluşturma

Sınıfları birer DNA'ya benzetebiliriz. Çünkü biyolojide DNA, organizmanın şeklini, özelliklerini belirleyen birer taslak görevi gören yapılardır. Aynı durum sınıflar içinde geçerlidir. Biz bir Class yani sınıf oluşturduğumuzda aslında birer canlı yapmış oluruz. İşte bu canlıyı kullanmak için ilk önce o canlıyı oluşturmak gereklidir. İşte nesne oluşturma eylemi budur.
Örneğin, bir matematik sınımızı olsun ve içersinde dört işlem olsun. Daha sonra ana fonksiyonda canlımız durumunda olan matematik sınımızın bir nesnesini oluşturalım.
class Matematik // Matematik sınıfı
{
    public int Topla(int a, int b)
    {
        return a + b;
    }

    public int Cikarma(int a, int b) 
    {
        return a - b; 
    }

    public float Bolme(int a, int b)
    {
        return a / b;
    }

    public int Carpma(int a, int b)
    {
        return a * b;
    }
}

internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        Matematik MAT = new Matematik(); // Matematik sınımızdan bir nesne oluşturduk
        // Ve kodlarımızda kullandık

        Console.WriteLine(MAT.Topla(2,2)); // 4
        Console.WriteLine(MAT.Cikarma(2,5)); // -3
        Console.WriteLine(MAT.Bolme(4,2)); // 2
        Console.WriteLine(MAT.Carpma(5,2)); // 10
    }
}

Böylelikle her yerde operatörlerle işlem yapmamız gerekmeden, sadece nesnemizin fonksyonlarıyla işlem yaptık.

Static Kullanımı

Static, nesne oluşturmadan sınıfın içeriğine ulaşmamızı sağlayan bir anahtar kelimedir.
Static ve nesne oluşturma eyleminin arasındaki fark; Static direk öz sınıfın içeriğini ulaşırken, nesne oluşturma sınıfın bir kopyasıyla işlem yapmamızı sağlar.
Şimdi nesne oluşturma ile yaptığımız matematik sınıfızı birde Static kullanımı ile görelim. Yapmamız gerek şey, sınıfımızın ve fonksiyonlarımızın erişim belirteçleri ile türleri arasında static ifadesi koymak. Ve artık nesne oluşturmak zorunda değiliz.
static class Matematik // Matematik sınıfı
{
    public static int Topla(int a, int b)
    {
        return a + b;
    }

    public static int Cikarma(int a, int b) 
    {
        return a - b; 
    }

    public static float Bolme(int a, int b)
    {
        return a / b;
    }

    public static int Carpma(int a, int b)
    {
        return a * b;
    }
}

internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        // Nesne oluşturmaya gerek kalmadan fonksiyonlarmıza direk erişim sağlayabiliriz

        Console.WriteLine(Matematik.Topla(2,2)); // 4
        Console.WriteLine(Matematik.Cikarma(2,5)); // -3
        Console.WriteLine(Matematik.Bolme(4,2)); // 2
        Console.WriteLine(Matematik.Carpma(5,2)); // 10
    }
}
