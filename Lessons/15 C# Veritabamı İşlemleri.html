Veritabanı İşlemleri

Veritabanı Nedir?

Veritabanları, verilerin bir bütün olarak saklandığı sistemlerdir.
Veritabanları genelde sunucularda, kullanıcıların verilerini saklamak için kullanılır. Kullanıcı adları, şifreleri gibi veriler saklanır. Veritabanlarında bir sorgu dilin olan SQL kullanılır.

SQL Komutları

SQL bilmeden veritabanlaryıla işlemler imkansızdır. Bu yüzden Data's Book on SQL eğitimime bakınız.
Aynı zamanda SQL dilimiz bir GUI arayüzünü görüntülücek bir araca da ihtiyacımız var. Onunda uyumlulukları sayesinde MariaDB ve HeidiSQL ile yapacağız. Bu yüzden Data's Book on MariaDB bakınız.

MariaDB Veritabanına Bağlantı Kurma

C# ile MariaDB'yi bağlanamadan önce indirmemiz gerekbir kütüphane var. İlk olarak Visual Studio'muzda NuGet paket yöneticisine gidelim. Arama kutusuna "MySqlConnector" yazalım ve ilk çıkanı indirip projemize dahil edelim.
Bağlantı için, veritabanımızın adresi gereklidir. MariaDB'den bir veritabanı oluşturalım ve bağlantı adresini kopyalalım. Ayrıca Using'ide bu kısımda kullanabiliyoruz. Ardından C# tarafında;
using System;
using MySqlConnector; // Gerekli kütüphanemiz. Çünkü MariaDB MySQL kullanıyor

namespace DatasBookonCsharp
{
    internal class Program
    {
        public static void Main(string[] args)
        {
            string conc = 
                "Server=127.0.0.1;" + // Sunucumuzun adresi
                "Port=3306;" + // Portu
                "Database=users;" + // Kullanıcağımız Database
                "User=root;" + // Yetkimiz
                "Password=****;"; // Database şifremiz

            using (MySqlConnection c = new MySqlConnection(conc)) // Bağlantı metni ile MySQL bağlantısı kur
            {
                c.Open(); // Kurulan bağlantıyı aç
            }
        }
    }
}

Hata vermediği taktirde bağlantı sorunsuz olmuş demektir.

Veri Ekleme (INSERT)

İlk önce verimizi ekliceğimiz tabloya değişkenlerimizi girelim. Ben name ve pass yaptım. Çünkü onların üzerinden bir işlem yapacağım. Şimdi kullanicilar'a, kullanıcıdan gelen verileri ekliyelim.
string conc = 
    "Server=127.0.0.1;" + // Sunucumuzun adresi
    "Port=3306;" + // Portu
    "Database=users;" + // Kullanıcağımız Database
    "User=root;" + // Yetkimiz
    "Password=****;"; // Database şifremiz

using (MySqlConnection c = new MySqlConnection(conc)) // Bağlantı metni ile MySQL bağlantısı kur
{
    c.Open(); // Kurulan bağlantıyı aç

    Console.Write("Kullanıcı isminiz: ");
    string username = Console.ReadLine();

    Console.Write("Şifreniz: ");
    string pass = Console.ReadLine();

    string qry = $"INSERT INTO kullanicilar (name, pass) VALUES ('{username}', '{pass}')"; // SQL Sorgumuz

    MySqlCommand cmd = new MySqlCommand(qry, c); // Sorgumuzu oluşturur 
    cmd.ExecuteNonQuery(); // Sorgumuzu çalıştırır
}
sonucunda
Veri eklenmiiş oldu lakin, bu sistemde SqlInjection yani Sql sorgusunu kötü niyetli yazılma olasılığı var. Bunun için kodumuzu şu şekilde değiştirelim.
string conc = 
    "Server=127.0.0.1;" + // Sunucumuzun adresi
    "Port=3306;" + // Portu
    "Database=users;" + // Kullanıcağımız Database
    "User=root;" + // Yetkimiz
    "Password=****;"; // Database şifremiz

using (MySqlConnection c = new MySqlConnection(conc)) // Bağlantı metni ile MySQL bağlantısı kur
{
    c.Open(); // Kurulan bağlantıyı aç

    Console.Write("Kullanıcı isminiz: ");
    string username = Console.ReadLine();

    Console.Write("Şifreniz: ");
    string pass = Console.ReadLine();

    string qry = "INSERT INTO kullanicilar (name, pass) VALUES (@name, @pass)"; // SQL Sorgumuz

    MySqlCommand cmd = new MySqlCommand(qry, c); // Sorgumuzu oluşturur 

    cmd.Parameters.AddWithValue("name", username); // Sorgunun @name ile işaretlenen kısma kullanıcının girdisini koy
    cmd.Parameters.AddWithValue("pass", pass); // Sorgunun @name ile işaretlenen kısma kullanıcının girdisini koy

    // Sorgu güvenli hale getirildi. SqlInjection saldırı önlendi

    cmd.ExecuteNonQuery(); // Sorgumuzu çalıştırır
}

Veri Sorgulama (SELECT)

SELECT, tablolardaki verileri çekmemizi sağlar.
Aynı örnekten gideceksek, kullanıcının girdiği bilgiler veritabanından çekerek ona iletsin
string conc = 
    "Server=127.0.0.1;" + // Sunucumuzun adresi
    "Port=3306;" + // Portu
    "Database=users;" + // Kullanıcağımız Database
    "User=root;" + // Yetkimiz
    "Password=****;"; // Database şifremiz

using (MySqlConnection c = new MySqlConnection(conc)) // Bağlantı metni ile MySQL bağlantısı kur
{
    c.Open(); // Kurulan bağlantıyı aç

    Console.Write("Kullanıcı isminiz: ");
    string username = Console.ReadLine();

    Console.Write("Şifreniz: ");
    string pass = Console.ReadLine();

    string qry = "INSERT INTO kullanicilar (name, pass) VALUES (@name, @pass)"; // SQL Sorgumuz

    MySqlCommand cmd = new MySqlCommand(qry, c); // Sorgumuzu oluşturur 

    cmd.Parameters.AddWithValue("name", username); // Sorgunun @name ile işaretlenen kısma kullanıcının girdisini koy
    cmd.Parameters.AddWithValue("pass", pass); // Sorgunun @name ile işaretlenen kısma kullanıcının girdisini koy

    cmd.ExecuteNonQuery(); // Sorgumuzu çalıştırır

    Console.Write("\n"); // Ayırma satırı

    string qry2 = "SELECT name, pass FROM kullanicilar WHERE name=@name"; // Sorgumuz. Sorgumuz bize sadece kullancının girdiği kullanıcı ismindeki sütünündaki verileri getirecektir

    cmd = new MySqlCommand(qry2, c); // Sorgumuzu çalıştırabilir hale getiriyoruz
    cmd.Parameters.AddWithValue("name",username); // Gerekli kısımları dolduruyoruz

    using (MySqlDataReader msdr = cmd.ExecuteReader()) // DataReader ile dönen veriyi okuyoruz
    {
        while (msdr.Read()) // Sırayla oku
        {
            Console.WriteLine("Name: " + msdr[0]); // Name'i yazdır
            Console.WriteLine("Password: " + msdr[1]); // Password'u yazdır
        }
    }
}
