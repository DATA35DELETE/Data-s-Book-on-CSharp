Native Dosya Oluşturulmas ve Kullanımı

Native Dosya Nedir?

Native veya yerli dosya assasen o yazılım dilinden olmayan ve başka bir yazılım dilinden yazılmış, kütüphane görevi gören dosya formatlarınıdır.
Native dosyaların kullanmasının başlıca nedenlerinden bir tanesi de tüm dillerin farklı kullanım ve uzmanlık alanları olduğundan, Native dosyalar ile diller arası bir iletişimi sağlar. Örneğin Java ile bir JavaFX uygulamsı yazıyor olun. Günümüz şartlarında JavaFX uygulamaların görünümü gayet iyi. Fakat Java bir sanal makinede çalıştığından dosya işlemleri Low-Level şekilde yapılamaz. İşte bu yüzden C veya Rust ile yazılan bir dosya işleme sistemi kodlarız ve onu işletim sistemimize göre bir Native dosya çıktısı alır ve JNI ile iletişimizi kurarız.
Genel olarak hibrid mimariler yani tek bir dile bağımlı kalmadan büyük projeler geliştirebilir olur. Ayrıca FFI yani C ile uyumluluk denilen bir ölçek vardır. Bu ölçek ne kadar yüksek ise o kadar Native dosya formatı kullanabirliği artar. Bunun nedeni ise neredeyse tüm dillerin temelinde C yatar.

P/Invoke

C# 'daki Native dosyaları kullanımı sağlayan bir kütüphanedir.
P/Invoke ile Native dosyaların içeriğine erişebiliriz. Dikkat edilmesi gereken şöyle bir durum vardır ki C# Marshal adı verilen bir String sistemi kullanıyor. Bu yüzden eğer bir Native dosyasından String çıktısı aldıysak Marshal.PtrToStringAnsi(ptr) yapmamız gerekir.
Aşşağıda örnek bir Rust kodu ile yaptığım bir fonksyionu C#'da çağrıcağım.
use std::ffi::CString;
use std::os::raw::c_char;

#[no_mangle]
pub extern "C" fn HelloWorld() -> *const c_char {
    let s = CString::new("Hello World").unwrap();
    s.into_raw()
}

Rust kodumuz. Cargo.toml da lib olarak tanımlıyoruz ve cargo build --release ile .dll dosyamızı yani Native dosyamızı elde ediyoruz. Ardından C# projemizde ana dizindeki bin/debug/net10.0 yolunu takip edip dosyamızı oraya atıyoruz.
using System.Runtime.InteropServices;

namespace DatasBookonCsharp
{
    internal class Program
    {
        [DllImport("nativeFileExample.dll", CallingConvention = CallingConvention.Cdecl)] // Native dosya ismimiz
        private static extern IntPtr HelloWorld(); // Rust daki fonksiyonumuz

        public static void Main(string[] args)
        {
            var ptr = HelloWorld(); // Fonksiyonu çağırıp çıktıyı alıyoruz

            Console.WriteLine(Marshal.PtrToStringAnsi(ptr)); // Dönen değeri C#'ın kullanabileceği şekilde çevirip yazdırıyoruz
        }
    }
}

C#'da Native Dosya Oluşturulması

Rust'da yazdığımız kodu C#'da yazalım.
using System.Runtime.InteropServices;
using System;
    
public class Program
{
    [UnmanagedCallersOnly(EntryPoint = "HelloWorld")]
    public static IntPtr HelloWorld()
    {
        return Marshal.StringToHGlobalAnsi("Hello World"); // Okunması için ANSI'e çeviriyoruz
    }
}

Visual Studio içerisinde bulunduurduğu projeAdi.csproj adındaki dosyanın içeriğine ise;

<Project Sdk="Microsoft.NET.Sdk">
	<PropertyGroup>
		<TargetFramework>net8.0</TargetFramework>
		<PlatformTarget>x64</PlatformTarget>
		<PublishAot>true</PublishAot>
		<NativeLib>Shared</NativeLib>
	</PropertyGroup>
</Project>


Ardından bir terminal açalım ve 
dotnet publish -c Release -r win-x64

Derleme tamamlandıktan sonra bin/release//8.0net/win-x64/publish yolunda Native dosyamız olmalıdır.
