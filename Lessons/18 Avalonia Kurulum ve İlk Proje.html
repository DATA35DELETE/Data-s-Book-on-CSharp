Kurulum ve İlk Proje

Eklenti Paketini İndirme

Avalonia kullanabilmemiz için ilk önce eklenti paketini indirmemiz gerekir. Bunun için Visual Studio'muzu açalım ve "Kodsuz olarak devam et" tıklayalım. Ardından ana ekranda "Uzantılar > Uzantıları Yönet" yolunu izliyelim. Ardından gelen menüde "Avalonia" aratalım. Avalonia ikonlu uzantıyı "İndir" butonuna basalım ve Visual Studio'yu kapatalım. İndirmeden ardından artık Avalonia'yı kullanabiliriz. Normal proje oluşturduğumuz gibi bu sefer proje

Proje Yapısı

Projemiz açıldığında bizi karşılıcak pencere MainWindow.axaml kod dosyasıdır. Bu kod dosyası AXAML tabanlı olan kod dosyamızdır. Solda görmüş olduğumuz üzere yazılı kodlar, sağda ise kodlarımızın ön gösterimi yer almaktadır.
Bir diğer önemli olan kısım ise MainWindow.axaml.cs dosyasıdır. Bu kısım ise MainWindow.axaml dosyasının Backend'didir. Yani arkada çalışan kodlardır.
            
İlk açılıştaki kodlar

using Avalonia.Controls;

namespace DatasBookonAvalonia // Ana proje ismimiz
{
    public partial class MainWindow : Window // Ana sınıfımız
    {
        public MainWindow() // İlk çalışan fonksiyon
        {
            InitializeComponent(); // Bizim MainWindow.axaml dosyasını bir pencereye dönüştür ve görüntüle
        }
    }
}
