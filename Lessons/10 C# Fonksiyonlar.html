Fonksiyonlar

Fonksiyon Nedir?

Fonksiyonlar, program akışını yönlendirebile ve çağrıldığında, tanımlanmış kodları çalıştıran yapılardır.
Fonksiyonlar genelde eğer bir iş birden fazla kodda uygulanıyorusa, kod tasarufu için fonksiyonlar kullanılır. Öğrenğin; Bir hesap makinesinde tüm işlemler ana fonksiyonda yani Main fonksiyonunda değil, her bir işlem bir fonksiyon şeklinde ihtiyaç durumunda çağrılır.
Aynı zaman erişim belirteçleri, veri türleri gibi kavramlarlada kullanılabilir.

Main Fonksiyonu

Main fonksiyonu, programlar için çok önemlidir. Programlar için başlama noktasına benzetebilceğimiz bu fonksiyon, ilk kodları çalıştırır ve programın çalışma başlangıcını sağlar.

Void'li Fonksiyonlar

Void, hiçlik demek. Void'li fonksiyonlar ise geriye hiçbir değer döndürmeyen fonkiyonlardır. Yani sadece program akışını değiştirirler.
internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        SayHello();
    }

    public static void SayHello()
    {
         Console.WriteLine("Hello, World!");
    }
}

Return'lü fonksiyonlar

Return'lü fonksiyonlar, Void'lilerin aksine geriye bir değer döndürürler. Bu değer döndürme olayını, matematikteki fonksiyonlar yapısına benzetebiliriz. Farkı; Matematikte sayılar alabilrken yazılım da sayı, yazı ve koşul ifadelerini alabiliyoruz.
internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        Console.WriteLine(HelloWorldText()); // HelloWorldText ile "Hello, World!" yazısını alma
    }

    // Hangi tür veri döndürülecekse, onun türü fonksiyon isminden önceki kısma yazılır
    public static string HelloWorldText()
    {
         return "Hello, World!";
    }
}

Parametreli Fonksiyonlar

Parametreli fonksiyonlar, çağrılırken bir veya birden fazla veri isteyebilen fonksiyonlardır. Parantezlerinde istenir ve çağrılırken parantezlerinde gönderilir.
internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        Console.WriteLine(KareAl(2)); // 4
    }

    public static int KareAl(int x) // Main'den gönderilen 2, x adlı bir değişkene atandı
    {
         return x * x;
    }
}
İstersek varsayılan değer de hatayabiliriz. Böylelikte parantezlerinde hiç bir şey göndermesek bile hata almadan işlemler gerçekleşebilir.
internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        Console.WriteLine(KareAl()); // 1
    }

    public static int KareAl(int x = 1) // Varsayılan x'in değeri 1
    {
         return x * x;
    }
}

Aşırı Yükleme

Aşırı yükleme, aynı isimle fonksiyonlar, farklı parametre almasıyla oluşan avantaja denir.
internal class Program
{
    static void Main(string[] args) // Ana fonksiyon
    {
        Console.WriteLine(Topla(2,2)); // int parametreli metot çağrısı
        Console.WriteLine(Topla(2.5f, 3.5f)); // float parametreli metot çağrısı
    }

    public static int Topla(int a, int b)
    {
         return a + b;
    }

    public static float Topla(float a, float b)
    {
        return a + b;
    }
}
