Dosya İşlemleri

Dosya İşlemleri Nedir?

Dosya işlemleri, dosyalar veya klasörler üzerinde işlemler yaptığımız işlemlerdir.
C# programlama dilinde File sınıfı ile dosya ve Directory sınıfı ile klasör işlemleri için kullanılır.

Dosya Oluşturma

File.Create("DosyaYolu.uzantisi"); // Dosya oluşturma

File.Copy("KaynakDosyaYolu","HedefDosyaYolu"); // Dosya kopyalama

File.Move("TaşınanDosyaYolu","HedefDosya"); // Dosya taşıma

File.Delete("DosyaYolu.uzantisi"); // Dosya silme

Directory

Directory.CreateDirectory("KlasorYolu"); // Klasör oluşturma

bool isDirectoryCreated = Directory.Exists("KlasorYolu"); // Klasör var mı diye kontrol eder. Bool değeri döndürür

Directory.Move("KaynakKlasör", "HedefKlasör"); // Klasör taşıma

string[] KlasorunIcindekiDosyalar = Directory.GetFiles("Klasör"); // Klasörün içersindeki dosyaları alır. String dizesi döndürür

Directory.Delete("Klasör"); // Klasörü siler

Dosya Yazma

Dosya yazma işlemi için StreamWriter kullanılır. Ayrıca bu kısımda using'in başka bir kullanım alanı görüceksiniz. Normalde görevi projeye kütüphane dahil etmedir. Lakin dosya işlemlerinde otomatik kapatma ve içersine yazmak için kullanılıyor. Normal şartlarda StreamWriter ile yazdığınız metinler, gerçek dosyaya işlemez. Bunun için .flush() komutu kullanılır. Using sayesinde bu koddan kurtulmuş olduk.
using (StreamWriter sw = new StreamWriter("dosyaYolu", true)) // True ile dosyanın içeriğine eklenir
{
    sw.WriteLine("Metin");
}

using (StreamWriter sw = new StreamWriter("dosyaYolu", false)) // False ile dosyanın içeriğinin üzerine yazılır.
{
    sw.WriteLine("Metin");
}

Dosya Okuma

Dosya okuma işini ise StreamReader sınıfı ile yapılır. Using aynı görevi üstlenerek işimizi kolaylaştırır.
string dosyaIcerigi = ""; // Dosya içeriği değişkeni
using (StreamReader sr = new StreamReader("dosyaYolu")) // StreamReader ile okuma işleminii başlatıyoruz
{
    string satir; // Her bir satır için değişken
    while ((satir = sr.ReadLine()) != null) // Her satırı null olmadığı süreççe oku ve diğerine geç
    {
        dosyaIcerigi += satir; // Satır içeriğini değişkenimize ekle
    }
}

Console.WriteLine(dosyaIcerigi); // Dosya içeriğimizi yazdır

JSON İşlemleri

JSON Nedir?

Verileri seriyalez etmek için kullanılan bir dosya formatıdır.
Using System.Text.Json; ile projemize dahil edelim.
JSON genel verileri bir dosya şekilde saklamak ve düzenlemek için kullanılır. Eğer büyük projeler ile uğraşıyorsanız JSON öğrenmek, kodunuzu daha performanslı yazmanızı sağlayabilir.

JSON Formatı Elde Etme

JSON formatı elde etmek için JsonSerializer.Serialize(Class); ve formatını elde etmek istediğimiz bir sınıf gereklidir.
Ufak bir örnek ile açıklayalım. Bir kullanıcı sınıfı ve onundan bir JSON formatı elde edelim.
class Users // Formatını elde edeceğimiz sınıf
{
    public string name { get; set; }
    public uint id { get; set; }
    public uint age { get; set; }
}


public static void Main(string[] args)
{
    Users usr = new Users(); // Nesnemizi oluşturuyoruz

    // Kullanıcıdan veri isteme
    Console.Write("İsminizi giriniz: ");
    string Name = Console.ReadLine();

    Console.Write("Id giriniz: ");
    string IdText = Console.ReadLine();
    uint Id = Convert.ToUInt32(IdText);

    Console.Write("Yaşınızı giriniz: ");
    string AgeText = Console.ReadLine();
    uint Age = Convert.ToUInt32(AgeText);

    // Verileri nesneye işleme
    usr.name = Name;
    usr.id = Id;
    usr.age = Age;

    string format = JsonSerializer.Serialize(usr); // Formatı elde etme

    Console.WriteLine("\nFormat: "+ format); // Konsola yazdırma
}

JSON Dosyası Okuma

JsonSerializer.Serialize(); sınıfları yazdırdığından, tersi olan Sinif sinif = JsonSerializer.Deserialize<Sinif>("metin") ile, JSON yazılırken kullanılan sınıf ve JSON dosyasının içeriğini alan bir String ifadesi ile JSON dosyasını okur.
Şimdi geliştirdiğimiz projeye birde JSON okuma ekleyelim.
class Users // Formatını elde edeceğimiz sınıf
{
    public string name { get; set; }
    public uint id { get; set; }
    public uint age { get; set; }
}


public static void Main(string[] args)
{
    Users usr = new Users(); // Nesnemizi oluşturuyoruz

    // Kullanıcıdan veri isteme
    Console.Write("İsminizi giriniz: ");
    string Name = Console.ReadLine();

    Console.Write("Id giriniz: ");
    string IdText = Console.ReadLine();
    uint Id = Convert.ToUInt32(IdText);

    Console.Write("Yaşınızı giriniz: ");
    string AgeText = Console.ReadLine();
    uint Age = Convert.ToUInt32(AgeText);

    // Verileri nesneye işleme
    usr.name = Name;
    usr.id = Id;
    usr.age = Age;

    string format = JsonSerializer.Serialize(usr); // Formatı elde etme

    using (StreamWriter sw = new StreamWriter("dosya.json")) // StreamWriter ile format'ı yazdırma
    {
        sw.WriteLine(format);
    }

    Console.WriteLine("\nJSON dosyası yazıldı"); // Bilgi mesajı

    Console.WriteLine("\n\nYazılan JSON dosyası okunuyor...\n"); // Bilgi mesajı

    string jsonString = "";

    using(StreamReader sr = new StreamReader("dosya.json", true)) // StreamReader ile JSON dosyasını okuma
    {
        string satir;
        while ((satir = sr.ReadLine()) != null)
        {
            jsonString += satir; // Satırları jsonString'e ekleme
        }
    }

    Users okunanUsr = JsonSerializer.Deserialize(jsonString); // jsonString'i çevirme

    // Okunan JSOn dosyasındaki verileri yazdırma
    Console.WriteLine("Name: " + okunanUsr.name);
    Console.WriteLine("Id: " + okunanUsr.id);
    Console.WriteLine("Age: " + okunanUsr.age);
}
