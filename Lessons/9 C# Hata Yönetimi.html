Hata Yönetimi

Hata yönetimi, olması muhtemel olan hataları yönetmek ve program akışını değiştirmek için kullanılan yapılardır.
C# programlama dilinde eğer bir hata çıkar ise, bütün program çökebilir. Üstelik bu durum sadece işte WriteLine yerine WritLine gibi yanlış yazım yanı Sytax hataları olmayan hatalar. Genellikle kulanıcı kaynaklı olan hataları yönetmemiz gereklidir. Mesela biz kullanıcıdan Integer isteriz, o bize String verir. Haliyle program hata verek kapanır. İşte bu gibi durumlarda hata yönetimi kullanılır.

Try-Catch

C# programlama dilinde hata yönetimi için kullanılır.
while (true)
{
    try // Muhtemel hatanın çıkacağı kısım
    {
        Console.Write("Bir sayı giriniz: ");
        int sayi1 = Convert.ToInt32(Console.ReadLine()); // Çıkarması muhtemel hatalı kod
    }
    catch (Exception ex) // Hatanın türü
    {
        Console.WriteLine(ex.Message);
    }
}

Böylelike hata bile alınsa program çökmeyecektir.

Hata Fırlatma

Her bir hatanın kendine has bir mesajı vardır. Eğer istersek kendi hata mesajımızla bir hata fırlatabiliriz.
while (true)
{
    try
    {
        Console.Write("1 veya 2 tuşlayınız: ");

        string input1 = Console.ReadLine();

        if (input1 == "1")
        {
            Console.WriteLine("1'i seçtiniz.");
        }
        else if (input1 == "2")
        {
            Console.WriteLine("2'yi seçtiniz.");
        }
        else
        {
            throw new Exception("Geçersiz giriş yaptınız."); // Eğer kullanıcı 1 veya 2'yi seçmezse, özel bir hata oluşturduk ve fırlattık
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine(ex.Message);
    }
}
