Olay Nedir?

Kullanıcının GUI bileşenleriyle etkileşime geçmesini sağlayan yapılardır.
Örneğin bir butona basıldığında uygulamanın kapanmasını istersek, bir butonun özniteliklerine "Click" özniteliğini ekliyelim. İçine ise istediğimiz isimli bir fonksiyon tanımlıyoruz. Ve kod gelen fonksiyonun içerisine ise System.Environment.Exit(0); yazıyoruz. Artık butona bastığımızda uygulamamız kapanacaktır.
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        mc:Ignorable="d" d:DesignWidth="800" d:DesignHeight="450"
        x:Class="DatasBookonAvalonia.MainWindow"
        Title="DatasBookonAvalonia">

		<Button Content="Uygulamayı Kapat"
                Click="Exited"/>
</Window>

Kod tarafı

using Avalonia.Automation.Peers;
using Avalonia.Controls;

namespace DatasBookonAvalonia
{
    public partial class MainWindow : Window
    {
        public MainWindow()
        {
            InitializeComponent();
        }

        private void Exited(object? sender, Avalonia.Interactivity.RoutedEventArgs e)
        {
            System.Environment.Exit(0);
        }
    }
}
