Değişkenler

Değişkenler Nedir?

Değişkenler, programlarda geçici veya kalıcı olarak depolanan değerlerdir.
Örneğin; Bir banka giriş sistemi yazdığımızı düşünelim. O sistemde kullanıcının banka şifresine ihtiyacımız öyle değil mi? Eğer değişkenler olmasa idi, programcı sürekli kullanıcıdan şifresini istemek zorunda kalıcaktı. İşte değişkenler bu kısımda devreye giriyor. Değişkenleri, eğer birden fazla kısımda bir değeri sürekli kullanıcaksak bir değişken tanımlarız ve ihtiyaç durumunda çağrırız

Sayısal Değişkenler

Byte

Byte, C#'daki en küçük sayı türüdür. 8 bit uzunluğundadır. Sbyte isimli bir alt türü vardır.
byte yas = 15; // 0 ila 255 arasındaki sayıları alabilir
sbyte borc = -100; // -128 ila 127 arasındaki sayıları alabilir. Farkı eksiye düşebilir

Short

Short, 16 bit uzunluğundadır. Ushort isimki bir alt türü vardır.
short MO2000 = -2000; // -32,768 ila 32,767 arasında tam sayı alabilir
ushort kurulus = 1923; // 0 ila 65,535 arasındaki sayıları alabilir. Farkı sadece pozitif sayılardır.

Integer

Integer, C# programlama dilinde en çok kullanılan sayısal değişken türüdür. Ayrıca 16, 32 ve 64 bit uzunluğundaki versiyonları vardır. Fakat varsıyan değeri 32 bit uzunluğundadır. Uint isimli bir alt türü vardır.
int paraMiktarı = 1000000; // -2,147,483,648 ila 2,147,483,647 arasında tam sayı alabilir
uint izmirinNüfusu = 4493242; // 0 ila 4,294,967,295 arasında tam sayı alabilir

System.Int64 altmisDortBitlikSayi = -64; // 64 bit uzunluğundaki sayı tanımlamak istersek
System.UInt64 isaretsiz = 64; // İşaretsiz versiyonu

Long

Long, C#'daki en uzun tam sayı değişken türüdür. Aynı zamanda 64 bit uzunluğunda ve System.Int64 sınıfına aittir. Dolaylı yoldan bir Integer sayısıdır.
long dunyaNufusu = 8000000000; // -9,223,372,036,854,775,808 ila 9,223,372,036,854,775,807 arasında tam sayı alabilir
ulong isaretsiz = 8000000000; // İşaretsiz versiyonudur.

Float

Float, C#'da çokça kullanılan ve ondalıklı sayıları ifade etmemizi sağlayan bir sayısal değişken türüdür. Sayının sonunda her zaman "f" ibaresi olmalıdır. Noktadan önce 9 basamaklık sayı alabilir.
float boy = 1.60f; // ±1.5 x 10^45 to ±3.4 x 10^38 arasında tam sayı alabilir

Double

Double, hassas işlemlerde kullanılan sayısal veri türüdür. Noktadan önce 17 basamaklı sayı gelebilir.
double hassasIslemSonuc = 1315453575.13451312; // ±5.0 × 10^−324 to ±1.7 × 10^308 arasında tam sayı alabilir

Sözel Değişkenler

Char

Char, tek bir karakter tutan bir sözel veri tipidir.
char A = 'A';

String

String, çoklu karakter tutan bir sözel veri tipidir. Char ile ilişkisi ise birden fazla Char'ın birleşimi sonucu oluşan tip, String'dir.
string isim = "Data Delete"; // Boyutu UInt'ın sınırıdır.

Boolean

Boolean, sadece iki değer alabilir. True ve False yani açık ve kapalı yani bir ve sıfır. 1 bit uzunluğundadır.
bool dogruMu = false;
bool yanlisMi = true;

İsimlendirme Kuralları

Değişkenler isimlendirilken bunlara dikkat edilmesi gereklidir. Aksi takdirde hatalar ile kaşılaşabilir.
Kural İsmi Açıklaması Yanlış ❌ Doğru ✅
Rakam Kullanımı Rakamları, değişkenin başına konulamaz. Lakin harflerden sonra kullanılabilir int 2000 = 2000; int s2000 = 2000;
Özel İşaretler Kullanımı Özel işaretler değişken tanımlarken kullanılmaz string $ = dolar; -
Boşluk Kullanımı İsimlendirirken boşluk kullanılmaz. Fakat _ kullanıla bilir char A Harfi = 'A'; char A_Harfi = 'A';
Case Kavramı Değişkenler kolay anlaşılsın diye Case'ler oluşturulmuştur. C#'da değişken isimlendirirken PascalCase kullanılır. Yani her kelime bitişik ve ilk harfleri büyük. Yapılmaması durumunda hata vermez byte yemekmasasininayaklari = 4; byte YemekMasasininAyaklari = 4;
Türkçe Karakter Kullanımı Türkçe karakterler C#'da hata çıkarmaz. Lakin yapısı gereği kullanılması önerilmez boolean Doğru = true; boolean Dogru = true;

Erişim Belirteçleri

Erişim belirteçleri, bir yapının kimlerin erişip erişemediğini belirleyen bir ifadedir. Erişim belirteçli değişkenler tanımlamak için Internal Class içine tanımlanır.
Erişim Belirteci Açıklaması Kullanımı
Private Sadece kendi ve alt Class'larca erişebilir private string sifre = "psswrd";
Protected Sadece kendi ve alt Class'larca erişebilir ve Class'dan türetilen Class'larca protected string adres = "İzmir";
Public Tüm Class'larca erişebilir public string ad = "Data";

Kullanıcıdan Veri Alma

Kullanıcıdan veri almak için Console.ReadLine() ile alınır. Ayrıca aldığı veri türü her zaman String'dir.
Console.WriteLine("Metin Giriniz:");
string girdi = Console.ReadLine();

Console.WriteLine($"Girilen metin: {girdi}"); // $ ve {} ile String birleştirilmesi yapılabilir

Değişlen Türleri Dönüştürme

Değişken tiplerini dönüştürmek için Convert kullanılır.

String → Integer

int sayi = Convert.ToInt32("1234"); //1234

Integer → String

string BesYuz = Convert.ToString(500); // "500"

Integer → Float

float derece = Convert.ToSingle(32); //32.0f

Float → Integer

int KusuratsizSayi = Convert.ToInt32(500.1213); // 500

Sabitlik

Sabitlik, bir değişkeni bir kere tanımladıktan sonra bir daha asla değiştirilemeyen verilerdir. İşlem const ifadesi ile uygulanır.
const float pi = 3.14;
pi = 2.34; // Hata alınacaktır.

Var

Var, değişken tiplerini otomatik olarak atayan bir anahatar kelimedir.
int sayi = 12; // Integer
var sayi2 = 3; // Integer
