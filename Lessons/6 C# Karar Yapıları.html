Karar Yapıları

Karar Yapıları Nedir?

Karar Yapıları, programın akışını yönlendirebilceğimiz yapılardır. Boolean değerlerine göre işlemlem yapmamızı sağlar.
Program akşını, C# programlama dili ve diğer benzeri dillerde kodlar yukarıdan aşağı doğru işlenir ve çalıştırıl. İşte bu sürece program akışı denir.
Örneğin; Bir banka uygulamsında kullanda, kullanıcının girdiği şifrenin gerçekten doğru olup olmadığını kontrol etmek için kullanırız. Eğer karar yapıları olmasaydı bütün programlar kullanılamaz hale gelir, hatta yapay zeka yapmak imkansız hale gelirdi.

If-Else If-Else

If, parantezlerindeki değer True ise kendi kod bloklarını çalıştırır. Else If, parantezlerindeki değer True ve If çalışmaz ise kendi kod bloklarını çalıştırır. Else ise hiçbir karar yapısı çalışmaz ise kendisi çalışır.
Bir programda sınırsız If, eğer üstünde If var ise sınırsız Else If konulabilir. Else ise üstünde If var ise sadece o If'e ait ve bir tane konulabilir.
int a = 1;

if (a == 1) // A bir ise
{
    Console.WriteLine("a = 1");
}
else if (a == 2) // A iki ise
{
    Console.WriteLine("a = 2");
}
else // A bir veya iki değilse
{
    Console.WriteLine("a != 1 ve a != 2");
}

Switch-Case-Default

Yapısı If-Else If-Else'e benzerdir. Switch'in parantezlerine sorgulanacak değişken tanımlanır. Case'lerde değişkenin değerime göre çalıştırıl. Default ise hiç bir Case çalışmaz ise kendisi çalışır.
int a = 2;

switch (a) // Sorgulanacak değişken
{
    case 1: // A bir ise
        Console.WriteLine("a = 1");
        break; // Case1'in bitiş kısmı. Konulmaz ise diğer Case'leri çalıştırır

    case 2:// A iki ise
        Console.WriteLine("a = 2");
        break;

    default: // A bir veya iki değilse
        Console.WriteLine("a != 1 ve a != 2");
        break;
}
