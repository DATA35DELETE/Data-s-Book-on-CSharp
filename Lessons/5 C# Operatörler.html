Operatörler

Operatörler Nedir?

Opoeratörler, C# yazılım dilinde basit matematiksel işlemler, karşılaştırma ve verileri mantıksal olarak değerlendirmemizi sağlayan araçlardır.
Örneğin operatörler bir hesap makinesindeki işlemleri kolaylıkla yapabilmemizi sağlayan araçlardır. Toplama, çıkarma, bölme ve çarpma gibi işlemlerde kullanılr. Veya bir işlemin sonucunun sıfır mı yoksa başka bir sayımı olup olmadığına bağlı olarak da programlar yazılabilir.

Atama Operatörü

Atama operatörü, bir değişkene bir değer atamak için kullanılır. Sembolü eşittir(=) işaretidir. Matematikdeki eşittirin görevinin aksine, programlamadaki eşittir, sağındaki değeri solundaki değişkene atar. Atama işlemi bir nevi eşitlemektir.
short KurulusYili = 1923; // 1923 değerini "KurulusYili" adlı bir değişkene atamış olduk

Aritmatik Operatörler

Aritmatik operatörler, basit matematiksel işlemleri yapabilen operatörlerdir.

+

Toplama işlemini gerçekleştirir.
int Toplama = 2 + 2; // 4

-

Çıkarma işlemini gerçekleştirir.
int Cikarma = 3 - 2; // 1

*

Çarpma işlemini gerçekleştiri.
int Carpma = 4 * 2; // 8

/

Bölme işlemini gerçekleştirir.
int Bolme = 2 / 2; // 1

Aritmatik Atama Operatörleri

Bu operatör, aritmatik ile atama operatörünü birleştirerek bize kolaylık sağlar. Örneğin bir değişkene 2 sayı eklememiz gerekir. Normal şartlarda yapmamız gereken kod: int a = a + 2;. Pek tabi bu kod hem uğraştırıcı hemde pratik değil. İşte bu gibi durumlarda aritmatil atama devreye girer ve kodu şu şekilde yazabilir oluruz.
int a = 2;

a += 2; // 4

Böylelikle kodu daha pratik yazmış olduk.

Karşılaştırma Operatörleri

Karşılaştırma operatörleri; bir değeri, bir diğer değer ile karşılaştırarar sonucundan True veya False sonucunu veren işlemlerdir.

>

Büyük mü, işlemini gerçekleştirir.
bool Islem = 2 > 3; // False

<

Küçük mü, işlemini gerçekleştirir.
bool Islem = 2 <  3; // True

>=

Büyük veya eşit mi, işlemini gerçekleştirir.
bool Islem = 2 >= 2; // True

<=

Küçük veya eşit mi, işlemini gerçekleştirir.
bool Islem = 3 <= 2; // False

==

Eşit mi, işlemini gerçekleştirir.
bool Islem = 2 == 2; // True

!=

Farklı mı, işlemini gerçekleştirir.
bool Islem = 23 != 2; // True

Mantıksal Operatörler

Mantıksal operatörler, bir Boolean değeri ile başka bir Boolean değerini karşılaştırarak bir True veya False sonucunu veren operatörlerdir.

!

Durumu tersine çevirir. Yani değer; True ise False'a, False ise True değerine çevirir.
bool dogru = true;

bool yanlis = !dogru; // "dogru"'nun değeri True olduğu için, "yanlis"'in değeri False'tır

&&

Ve mantık kapı görevini gerçekleştirir. Aşşağıdaki tabloya göre de değeri belirlenir. Tablodaki birler True'yu, sıfırlar ise False'ı temsil etmektedir.
  A Değişkeni     B Değişkeni     Sonuçları  
0 0 0
1 0 0
0 1 0
1 1 1 
bool Islem = true && true; // True

bool Islem2 = true && false; // False

||

Veya mantık kapı görevini görürler. Aşşağıdaki tabloya göre de değeri belirlenir. Tablodaki birler True'yu, sıfırlar ise False'ı temsil etmektedir.
  A Değişkeni     B Değişkeni     Sonuçları  
0 0 0
1 0 1
0 1 1
1 1 1 
bool Islem = true || true; // True

bool Islem2 = true || false; // True

Ternary

Ternary, değerin True veya False olduğuna göre tanımlı değeri sonuç olarak veren bir operatördür.
Operatörü kullanmak için soru işareti(?) ve iki nokta(:) kullanılarak gerçekleştirir. Soru işaretinin solundaki değer True ise, iki noktanın solundaki değeri sonuç olarak verir; Soru işaretinin solundaki değer False ise, iki noktanın sağındaki değeri sonuç olarak verir.
string a = true ? "T" : "F"; // "T"

string b = false ? "T" : "F"; // "F"
Böylelikle çok yönlülük özelliğini sağlar. Koşullara göre değerleri değiştirir.

Yorum Operatörleri

Yorum operatörleri, makineler için hiçbir ifade etmez iken, insanlar için; O kodun hangi işleve sahip olduğunu anlatmamızı sağlayan operatörlerdir. Çift eğik çizgi(//) ile yapılır.
int a = 12; // Bu değişkenin çoğu kısımda kullanılır
        
Şeklinde kullanılır. Böylelikle yazımlımcılar arasında kodlarını anlatmak için hayati önem taşır.
