Döngüler

Döngüler

Döngüler, kendini sürekli veya belirli bir miktar kadar tekrarlanan yapılardır. Döngüler, program akışınıda engeller. Bu sebeble bir döngü çalışır iken diğer kodlar, döngünün bitmesi beklerler.
Genel olarak; Kod tekrarlarını engeller. Örneğin kullanıcıya yüz kere Console.WriteLine("Hoşgeldin"); mesajı yazdırmak istiyelim. Peki yüz satır bu komutumu yazacağız. Tabiki hayır!
for (int i = 0; i <= 100; i++)
{
    Console.WriteLine("Hoşgeldin");
}

Böylelikle kodumuz yüz satırdan dört satıra düşmüş oldu.

While

While, parantezlerindeki Boolean değeri True olduğu süreççe kodlarını çalıştırır.
DİKKAT!!! : While döngüsünde kısır döngü ihtimali vardır. Kısır döngüler programı çökertebilir.
int a = 2;

while (a == 2) // Şartımız, a'nın değeri iki olduğu süreççe kodlarını çalıştırır
{
    Console.WriteLine("a = 2");
}

For

For, While gibi Boolean değere bakmaz. For, belirlenmiş miktar kadar kodlarını çalıştırır.
for (int i = 0; i <= 100; i++)
{
    Console.WriteLine("Hoşgeldin");
}
Yukarıda örneğini verdiğimiz kodda, For yüz kere kodlarını çalıştırmakta ve Konsola yüz kere "Hoşgeldin" yazdırmaktadır.

Foreach

Foreach, For döngüsünün bir alt türüdür. Bu yapı ise belirlenen liste elemanlarını her bir dönüşte, bir sonraki elemana geçen bir döngüdür. "Listeler" sayfasında daha detaylı anlatacağız.
