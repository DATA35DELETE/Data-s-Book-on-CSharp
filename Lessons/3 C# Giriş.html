Giriş

Yazılım Nedir?

Yazılım, donanıma elektriksel olarak hangi eylemi yapmasını gerektiği söyleyen bir dildir. Yani yazılım olmasaydı donanımlar pek işimize yaramazdı. Aynı şekilde donanım olmasaydıda yazılım var olamazdı.

Donanım Nedir?

Donanımda gözle görebildiğimiz dokunabildiğimiz, bilgisayarın içersinde bulununan nesnelerdir. Yazılım ise gözle görülemez veya dokunulamaz.
Yazılım dillerinin iki türü vardır.

Low-Level ve High-Level

Yazılım dilleri, derinliklerine bu iki kategoriye girerler. Low-Level yazılım dilleri, makine dili olan, meşur birler ve sıfırler, Binary'ye yakın olanlardır. Bu diller insanlar tarafından zor okunur. High-Level dilleri ise insanlar tarafından kolaylıkla okunabilir fakat makineler tarafından zor okunur. Bu High ve Low mesela sanılanın aksine dillerin güçlerinin değil, insan diline ne kadar yakın olduğunu belirtir.

Yazılım Dilleri

Rust, C/C++, Java, Kotlin, Python. Bunlar duyabilmiş olcağınız yazılım dilleridir.

C# Nedir?

Microsoft tarafından geliştirilen, .NET projesi için yapılan bir programlama dilidir.

Tarihçesi

C#, 2000 yalında Anders Hejlsberg'ın liderliğiyle beraber Microsoft tarafından geliştirilmiştir. Dil yapılırken Java'nın söz dizimini ve C ve C++'ın gücünü birleştirilmeye çalışılmıştır. Söz dizimi ise yazılımın birnevi yazılma kuralı. Dil, nesne yönelimli programlamayı merkeze alır. İlk sürümü 2002 de çıkmış olup, 2025 tarihinde son kararlı sürümünü yayımlamıştır. Ayrıa son sürümlerde, Python söz dizimine benziyecek şekilde Class'sız programlama yapabilceğiniz bir hale gelmiştir. Fakat biz Class'lı halini işliyeceğiz.

C# Nasıl Kullanılır?

C# kullanabilmek için yani yazabilmek için ilk önce onu yazabilciğimiz bir ortama ihtiyacımız var. İşte onuda Visaul Studio ile sağlayacağız. Tabi Visaul Studio biraz karmaşık ve ağır. O yüzden hem Visual Studio kurulumunu hem de konsol üzerinden Visual Studio Code ile nasıl kullanabilceğinizi göstereceğim. Kodlarda herhangi bir değişiklik olmucaktır, her iki ortamda da çalışıcaktır.

OOP

Geldik C#'ın olmassa olmazı OOP yani Nesne Yönelimli Programlama, OOP aslında bizlere; Kalıtım, çok biçimlilik, soyutlama ve kapsülleme gibi özellikler sunar. Bunlar kod tekrarlarını önleyerek daha iyi ve performanslı uygulamalar geliştirmemizi sağlar. İlerleyen sayfalar uygulamalı şekilde OOP daha derin işliyeceğiz

C#'ı Kurma

C# programlama dilini kurmak için Visual Studio'yu kullanıcağız.

Visual Studio Kurulumu

İlk olarak Visual Studio'nun sitesinden indirelim. İndirilen .exe dosyasını çalıştıralım ve kurulumu başlatalım. Ardından gelen ekranda kullanıcağımız SDK'leri indirelim.
Ekran görüntüsünde de gözüktüğü üzere C++, Python ve NodeJS gibi araçlarıda bu IDE'den kullanabiliriz.

Proje Oluşturma

Kurulum tamamlandıktan sonra Visual Studio'muzu açalım. Açılan pencerede "Yeni bir proje oluştur" butonuna basalım.
Ardından "Sonraki" butona basalım. Ardından gelen pencerede "Üst düzey deyimler kullanmayın" kutusunu işaretledikten sonra "Oluştur" diyerek projemizi oluşturalım.
Tebrikler! Artık C# uygulamaları yazabiliriz.

İLk Kod

Kodlarımızı yazmadan önce şu komutu göstermek isterim.
class Program // Ana Class
{
    static void Main() // İlk çağrılan fonksiyon
    {
        Console.WriteLine("Merhaba Dünya!"); // Konsola yazı yazdırma
    }
}

Bu programı çalıştırcak olursak, ekranda "Merhaba Dünya!" yazısını görürüz. Bundan şunu anlıyoruz ki Console.WriteLine(), ekrana bişiler yazdırmamızı sağlıyor.
